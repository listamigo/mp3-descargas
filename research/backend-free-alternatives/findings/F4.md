# F4: Frameworks open source para dashboard/admin

## Findings

### [1] AdminJS es un panel de administración open-source bajo licencia MIT para aplicaciones Node.js, con 4.3k estrellas en GitHub y ~4.2k descargas semanales en npm.
- quote: "AdminJS is an auto-generated admin panel for your Node.js application that allows you to manage all your data in one place. Always open-source under the MIT license."
- url: https://adminjs.co/
- source_type: primary
- published: unknown
- confidence: high

### [2] AdminJS se integra con Express, Hapi, Nest.js y Firebase mediante plugins nativos, y soporta adaptadores para Sequelize, TypeORM, Mongoose, Prisma y MikroORM.
- quote: "Thanks to the following Database Adapters, you can feed AdminJS with the data from almost any database."
- url: https://adminjs.co/
- source_type: primary
- published: unknown
- confidence: high

### [3] Retool ofrece un tier gratuito auto-hospedado limitado a 5 usuarios y 500 ejecuciones de workflows por mes, pero el software core es propietario y no open-source.
- quote: "Self-Host Plans: Free — Up to 5 users"
- url: https://retool.com/pricing
- source_type: primary
- published: unknown
- confidence: high

### [4] Directus es un backend source-available bajo la licencia Monospace Sustainable Core License (MSCL), gratuito para organizaciones con menos de $5M de ingresos anuales y 50 empleados.
- quote: "Free for most organizations. Under $5M annual revenue and 50 employees? Directus is completely free."
- url: https://github.com/directus/directus
- source_type: primary
- published: unknown
- confidence: high

### [5] Directus provee APIs REST y GraphQL instantáneas desde cualquier base de datos SQL, con un Studio visual, asistente de IA y servidor MCP nativo.
- quote: "Connect any SQL database and get instant REST and GraphQL APIs, a visual management Studio, and a native MCP server for AI agents."
- url: https://github.com/directus/directus
- source_type: primary
- published: unknown
- confidence: high

### [6] Strapi es un CMS headless open-source bajo licencia MIT con 72.6k estrellas en GitHub, ofreciendo un dashboard de administración personalizable y APIs REST y GraphQL auto-generadas.
- quote: "Strapi is an open-source, self-hosted headless CMS that lets developers build content APIs fast while giving content creators a friendly editing interface."
- url: https://github.com/strapi/strapi
- source_type: primary
- published: unknown
- confidence: high

### [7] Strapi soporta TypeScript, SQLite, PostgreSQL, MySQL, MariaDB, e incluye un sistema de plugins para extender funcionalidades.
- quote: "First-class TypeScript support with flexible database options (SQLite, PostgreSQL, MySQL, MariaDB)"
- url: https://github.com/strapi/strapi
- source_type: primary
- published: unknown
- confidence: high

### [8] AdminJS proporciona RBAC (control de acceso basado en roles) listo para usar, permitiendo control por recursos y acciones.
- quote: "AdminJS offers authentication and role-based access control (RBAC) out of the box."
- url: https://adminjs.co/
- source_type: primary
- published: unknown
- confidence: high

### [9] La implementación on-premise de Retool utiliza Docker Compose y soporta Kubernetes, con un repositorio open-source impulsado por la comunidad para scripts de despliegue.
- quote: "Deploying Retool On Prem"
- url: https://github.com/tryretool/retool-onpremise
- source_type: primary
- published: unknown
- confidence: medium

### [10] Directus soporta múltiples bases de datos incluyendo PostgreSQL, MySQL, MariaDB, MS SQL, SQLite, OracleDB y CockroachDB.
- quote: "Bring your own database: Postgres, MySQL, MariaDB, MS SQL, SQLite, OracleDB, CockroachDB, and more."
- url: https://github.com/directus/directus
- source_type: primary
- published: unknown
- confidence: high

### [11] El dashboard de administración de Strapi es personalizable, permitiendo a los desarrolladores reformar el frontend y extender la lógica backend mediante controladores, servicios y middleware.
- quote: "Go past plugins: write custom backend logic (controllers, services, middleware) and reshape the admin panel."
- url: https://strapi.io/
- source_type: primary
- published: unknown
- confidence: high

### [12] AdminJS se instala vía npm y funciona como middleware en servidores Node.js, requiriendo configuración mínima.
- quote: "Install it from NPM"
- url: https://adminjs.co/
- source_type: primary
- published: unknown
- confidence: high

## Dead ends
- Estado open-source de Retool: El producto core es propietario; solo los scripts de despliegue son open-source.
- Licenciamiento de Directus: La licencia MSCL es source-available pero no es open-source aprobada por OSI.

## Suggested follow-ups
- Comparar la facilidad de integración de cada framework con un backend Node.js/Express.
- Evaluar el tamaño de la comunidad y disponibilidad de soporte para cada opción.
- Analizar la escalabilidad y rendimiento para una app compartida de 10-50 usuarios.
