## Dead ends
- Supabase Realtime pricing page had no separate free-tier detail beyond what the main pricing page provided.
- Nhost self-hosted pricing was not included as it's separate from cloud pricing and the research focuses on cloud-hosted free tiers.

## Suggested follow-ups
1. For a 10-50 user MP3 Downloader app, which BaaS has the least restrictive "pause after inactivity" policy and how to keep it alive?
2. Can Firebase Firestore's 1 GiB + 50K reads/day limit handle analytics event logging for 50 concurrent users?
3. Supabase vs Nhost: which offers better Row Level Security for multi-user apps with Supabase's 500MB vs Nhost's 1GB database?
