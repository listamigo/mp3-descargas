# F2: Plataformas de hosting gratuitas con base de datos

## Findings

### [1] Render ofrece la mejor free tier real: servicios web + Postgres + Redis sin tarjeta de crédito
- quote: "You can deploy certain Render service types free of charge: Web services, Render Postgres databases, Render Key Value instances"
- url: https://docs.render.com/free
- source_type: primary
- published: unknown (current as of 2025-2026)
- confidence: high

### [2] Render free web service: 512 MB RAM, 0.1 CPU, $0/mes, pero se apaga tras 15 min sin tráfico
- quote: "Render spins down a Free web service that goes 15 minutes without receiving any inbound traffic. This process takes about one minute."
- url: https://docs.render.com/free
- source_type: primary
- published: unknown
- confidence: high

### [3] Render free Postgres: 256 MB RAM, 1 GB storage, pero EXPIRA a los 30 días
- quote: "Free Render Postgres databases expire 30 days after creation. An expired Free database is inaccessible unless you upgrade it to a paid instance type."
- url: https://docs.render.com/free
- source_type: primary
- published: unknown
- confidence: high

### [4] Render Hobby plan (gratis) incluye 750 instance hours/mes, 5 GB de bandwidth, soporta Node/Python/Docker
- quote: "Hobby — $0/month + compute. Deploy up to 25 services. 5 GB of bandwidth included."
- url: https://render.com/pricing
- source_type: primary
- published: unknown
- confidence: high

### [5] Railway no tiene free tier permanente: ofrece un crédito único de $5 sin tarjeta, después cobra $5/mes
- quote: "Free Trial — One-time $5 credit grant to try Railway. No credit card required."
- url: https://railway.com/pricing
- source_type: primary
- published: unknown
- confidence: high

### [6] Fly.io requiere tarjeta de crédito para TODAS las organizaciones, sin excepción
- quote: "All organizations (except for Linked Organizations) require a credit card on file."
- url: https://fly.io/docs/about/pricing/
- source_type: primary
- published: unknown
- confidence: high

### [7] Fly.io free trial: solo 2 horas de VM runtime o 7 días, whichever comes first — insuficiente para uso continuo
- quote: "A free trial on Fly.io includes 2 hours of machine runtime or 7 days of access, whichever comes first."
- url: https://fly.io/docs/about/free-trial/
- source_type: primary
- published: 2025-10-27
- confidence: high

### [8] Koyeb no tiene free tier para compute: el plan Pro cuesta $29/mes como mínimo
- quote: "Pro — $29/mo +compute. $10 included compute. 10 users."
- url: https://www.koyeb.com/pricing
- source_type: primary
- published: unknown
- confidence: high

### [9] Koyeb ofrece un tier "Free 5h" para Postgres (0.25 vCPU, 1 GB RAM) pero requiere plan Pro para correr
- quote: "Free 5h — $0/hr, $0/mo. vCPU 0.25, RAM 1 GB, Storage 1GB"
- url: https://www.koyeb.com/pricing
- source_type: primary
- published: unknown
- confidence: high

### [10] Cyclic.sh fue inaccesible durante la investigación (error de transporte DNS), información no disponible
- quote: (n/a)
- url: https://www.cyclic.sh/pricing
- source_type: n/a
- published: unknown
- confidence: low

### [11] Render free Postgres no soporta backups ni connection pooling en tier gratuito
- quote: "Free Render Postgres databases don't support any form of backups. Free Render Postgres databases don't support managed connection pooling."
- url: https://docs.render.com/free
- source_type: primary
- published: unknown
- confidence: high

### [12] Fly.io Managed Postgres no tiene tier gratuito; el unmanaged cuesta ~$2/mes para dev
- quote: "the cost for the preset configurations is about $2/month for a single node cluster for dev projects"
- url: https://fly.io/docs/about/pricing/
- source_type: primary
- published: unknown
- confidence: medium

## Dead ends
- Cyclic.sh: sitio web inaccesible (DNS resolution failed), no se pudo obtener información de pricing
- No se encontraron free tiers reales en Koyeb ni Fly.io que cumplan "sin tarjeta de crédito + uso continuo"
