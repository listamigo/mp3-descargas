# F3: Herramientas de analytics y monitoreo gratuitas

## Findings

### [1] PostHog Cloud free tier offers 1M events/month, 5K session replays, 1M feature flag requests, 100K exceptions, and 1500 survey responses per month — all without a credit card
- quote: "Free tier on all plans (resets monthly): Analytics 1M events, Session replay 5K recordings, Feature flags 1M requests, Error tracking 100K exceptions, Surveys 1500 responses"
- url: https://posthog.com/pricing
- source_type: primary
- published: unknown
- confidence: high

### [2] PostHog self-hosting is free under MIT license but requires significant infrastructure (4 vCPU, 16GB RAM, 30GB+ storage) and no paid support is offered
- quote: "PostHog is open-source and freely available for anyone to host themselves. We offer a free Docker Compose deployment under an MIT license... we don't offer any sort of guarantees around it working in certain ways on your infrastructure"
- url: https://posthog.com/docs/self-host
- source_type: primary
- published: unknown
- confidence: high

### [3] Sentry free "Developer" plan includes 5K errors/month, 5M spans tracing, 50 replays, and 5GB logs — but is limited to one user
- quote: "developer: For solo devs working on small projects. Free $0. Limited to one user. Error Monitoring and Tracing, Alerts and notifications via email, 10 custom dashboards, 5k errors, 5M spans, 50 replays"
- url: https://sentry.io/pricing/
- source_type: primary
- published: unknown
- confidence: high

### [4] Sentry self-hosted is available via Docker Compose under a "Fair Source" license (BSL), described as "feature-complete and packaged up for low-volume deployments"
- quote: "Sentry, feature-complete and packaged up for low-volume deployments and proofs-of-concept"
- url: https://github.com/getsentry/self-hosted
- source_type: primary
- published: unknown (latest release 26.6.0 as of Jun 2026)
- confidence: high

### [5] Plausible Community Edition (CE) is free self-hosted under AGPLv3, with basic bot filtering and no premium features (funnels, user journeys, ecommerce revenue, SSO, Sites API)
- quote: "Plausible Community Edition (CE) is the free, self-hosted, AGPL-licensed release... Marketing funnels, user journeys, ecommerce revenue goals, SSO and sites API are not available"
- url: https://plausible.io/self-hosted-web-analytics
- source_type: primary
- published: unknown
- confidence: high

### [6] Plausible Cloud starts at $9/month for 10K pageviews with 30-day free trial; no free tier exists on the managed service
- quote: "Up to 10k monthly pageviews. Starter $9/month... Sign up for 30-day free trial. No credit card required."
- url: https://plausible.io/#pricing
- source_type: primary
- published: unknown
- confidence: high

### [7] Umami v2 is fully self-hosted and free (MIT license), requiring only Node.js 18.18+ and PostgreSQL; Docker Compose deployment is supported out of the box
- quote: "Installing from source: Get the code from GitHub and build the application yourself... Docker compose: Build your own Docker container using docker compose"
- url: https://umami.is/docs/install
- source_type: primary
- published: unknown
- confidence: high

### [8] PostHog Cloud offers EU (Frankfurt) and US (Virginia) data residency regions, both available on the free plan
- quote: "Cloud region: US (Virginia) / EU (Frankfurt)"
- url: https://posthog.com/pricing
- source_type: primary
- published: unknown
- confidence: high

### [9] PostHog free tier drops events permanently once exceeded (unlike paid plans which bill overage); feature flags return default quota-limited responses
- quote: "On the free plan, any additional events are permanently dropped and feature flags will return a default quota limited response."
- url: https://posthog.com/pricing
- source_type: primary
- published: unknown
- confidence: high

### [10] Sentry self-hosted is described as "low-volume deployments and proofs-of-concept" — not recommended for production at scale without Sentry's managed infrastructure
- quote: "Sentry, feature-complete and packaged up for low-volume deployments and proofs-of-concept"
- url: https://github.com/getsentry/self-hosted
- source_type: primary
- published: unknown
- confidence: high
